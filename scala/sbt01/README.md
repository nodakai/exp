How to use SBT
==============

    $ cd (this directory)

    $ sbt compile
    $ sbt run

  * http://www.scala-sbt.org/release/docs/Getting-Started/Running.html

Continuous build (triggered execution)
--------------------------------------

    $ sbt
    > ~compile

  * http://www.scala-sbt.org/release/docs/Detailed-Topics/Triggered-Execution.html
